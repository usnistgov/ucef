Please use the above link to download UCEF 1.0.0 BETA rather than trying to build the virtual machine from github

Our automated build process no longer works due to recent changes to the distribution of the Oracle Java Development Kit (JDK). Our team is working to transition the UCEF virtual machine to an alternative, open-source implementation of Java called OpenJDK. Until we have a stable virtual machine based on OpenJDK, builds from github will be unavailable. Interested users should download the UCEF 1.0.0 BETA from the link above. A new version of the virtual machine that contains the current stable version of the code will be released on this site soon.

UCEF is an open-source tool kit that provides a collaborative experiment-development environment.

The National Institute of Standards and Technology (NIST) and its partner, the Institute for Software Integrated Systems at Vanderbilt University, have developed a collaborative experiment development environment across heterogeneous architectures integrating best-of-breed tools including programming languages, communications co-simulation, simulation platforms, hardware in the loop, and others. This environment combines these simulators and emulators from many researchers and companies with a standardized communications protocol, IEEE Standard 1516 High Level Architecture (HLA).
